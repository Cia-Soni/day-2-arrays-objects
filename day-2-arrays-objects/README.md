student mark list - Arrays & Objects

#project description
This mini project demonstrates the use of JavaScript arrays and objects to
store and display student marks. Data is shown both in the console and on a
simple UI.

---

#features
- Stores student data using objects
- Uses array methods like push, map, and filter
- Displays student marks dynamically on UI
- Console output for better understanding

---
#technologies used
- HTML5
- CSS3
- JavaScript (ES6)

---
#what I learned
- How arrays store multiple values
- How objects store structured data
- Using push() to add data
- Using map() to display data
- Using filter() to get specific results
- Connecting JavaScript with HTML UI
